CoolStar is a HVAC service comapny. This project is a comprehensive Next.js application that utilizes Prisma for database management and NextAuth for authentication. It provides a robust foundation for building complex web applications with a focus on security, scalability, and maintainability. The project includes features such as user authentication, booking management, and map integration.
